# ToM-Router Dataset

A benchmark for classifying whether social reasoning questions require Theory of Mind (ToM).

## What makes this dataset different

Most ToM benchmarks test whether a model *can do* Theory of Mind. This dataset tests whether a model can *recognize when* Theory of Mind is needed — the routing problem.

The dataset is specifically designed to be **shortcut-free**:
- A source-only classifier (knowing only which benchmark a sample came from) achieves **54.24%** — near random chance
- This is achieved through **contrastive augmentation**: for each story context, we generate both a ToM question and a non-ToM question, so the model must read the question rather than recognize writing style

| File | Samples | Description |

## Schema

| Field | Type | Description |
|-------|------|-------------|
| `sample_id` | string | Unique identifier (e.g., `simpletom_potato_chip_sev1_action`) |
| `source_dataset` | string | Origin dataset (e.g., `tomi_nli`, `social_iqa_contrastive`) |
| `context` | string | Story or dialogue context |
| `question` | string | Question about the context |
| `answer` | string | Gold answer (where available) |
| `requires_tom` | int | **Label**: 1 = requires ToM reasoning, 0 = does not |
| `subtype` | string | Finer category (belief, emotion, norm, relation, etc.) |
| `original_category` | string | Category from the source dataset |
| `split` | string | `train`, `val`, or `test` |
| `teacher_prob_tom` | float | OLMo-3-7B soft probability that ToM is required (0.0 to 1.0) |
| `teacher_label` | int | OLMo-3-7B binary decision (0 or 1) |
| `teacher_rationale` | string | OLMo-3-7B one-sentence explanation |
| `metadata` | dict | Source-specific metadata |

## Teacher Soft Labels

Every sample has a soft probability from OLMo-3-7B-Instruct (4-bit quantized):

| Teacher confidence | Count | Description |
|-------------------|-------|-------------|
| prob < 0.1 (confident non-ToM) | 3,517 | Teacher is sure this doesn't need ToM |
| 0.1 ≤ prob ≤ 0.9 (uncertain) | 4,232 | Borderline cases — the value of soft labels |
| prob > 0.9 (confident ToM) | 3,033 | Teacher is sure this needs ToM |

Teacher-ground truth agreement: 56.4% on original samples, 90.3% on contrastive samples.

## Contrastive Augmentation

2,855 samples were generated by OLMo-3-7B-Instruct to create opposite-label questions for existing contexts:

**Example — ToM story, generated non-ToM question:**
> **Context:** *"Isla entered the porch. The cucumber is in the red bathtub. Isla moved the cucumber to the red bottle. Chloe exited the porch."*
> **Original (ToM):** *"Where does Chloe think the cucumber is?"* — requires modeling Chloe's false belief
> **Generated (non-ToM):** *"Where was the cucumber after Isla moved it?"* — factual, no hidden beliefs

**Example — non-ToM story, generated ToM question:**
> **Context:** *"Jesse borrowed Jordan's car for the weekend to move apartments."*
> **Original (non-ToM):** *"What does Jesse need to do before this?"* — common sense
> **Generated (ToM):** *"What might Jordan believe about Jesse's intentions?"* — requires modeling Jordan's mental state

## Shortcut Verification

| Baseline | Accuracy | Interpretation |
|----------|----------|---------------|
| Random chance | 50.0% | — |
| Source-only classifier | 54.2% | Source shortcut eliminated |
| Context-length-only classifier | 52.8% | Length is not a shortcut |
| Bag-of-words classifier | 92.5% | Some lexical cues remain |

## How to Load

```python
import pandas as pd

df = pd.read_parquet("data/processed/router_dataset_hardened_v2.parquet")

train = df[df["split"] == "train"]
val = df[df["split"] == "val"]
test = df[df["split"] == "test"]

# Each sample has:
# - context, question: the input pair
# - requires_tom: binary label (0 or 1)
# - teacher_prob_tom: soft probability from OLMo-3 (0.0 to 1.0)
```
